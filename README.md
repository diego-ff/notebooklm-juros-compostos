# 📘 Miniguia de Estudo - Juros Compostos e Planejamento Financeiro

Este repositório contém o desenvolvimento de um caderno temático utilizando o **NotebookLM (Google)** como parte de um desafio prático de Engenharia de Prompt.

O objetivo foi reunir fontes abertas sobre um tema financeiro introdutório e utilizar IA como ferramenta de aprendizagem ativa, aplicando técnicas estruturadas de elaboração de prompts.

---

## 🎯 Tema

**Juros Compostos e Planejamento Financeiro**

---

## 📚 Objetivos de Estudo

- Compreender o conceito de juros compostos  
- Interpretar a fórmula matemática e seus componentes  
- Diferenciar juros simples e compostos  
- Analisar o impacto do crescimento exponencial no longo prazo  
- Aplicar simulações financeiras  
- Testar variações estruturadas de prompts  

---

## 🧠 Metodologia

O projeto foi desenvolvido no NotebookLM utilizando:

- Curadoria de 3 a 5 fontes abertas (PDF e textos)
- Definição de objetivos de estudo
- Elaboração de perguntas estratégicas
- Testes comparativos de prompts
- Registro das respostas com referências
