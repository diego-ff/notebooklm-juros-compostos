# Resumo Estruturado - Juros Compostos

## 1. Conceito

Juros compostos são calculados sobre o capital inicial acrescido dos juros acumulados ao longo do tempo.

## 2. Fórmula

M = C (1 + i)^t

Onde:
- M = Montante final
- C = Capital inicial
- i = Taxa de juros
- t = Tempo

## 3. Diferença entre Juros Simples e Compostos

Juros simples são calculados apenas sobre o valor inicial.
Juros compostos consideram juros sobre juros, gerando crescimento exponencial.

## 4. Aplicação Prática

Simulações financeiras demonstram que, no longo prazo, pequenas taxas de juros compostos produzem grande impacto no montante final.
